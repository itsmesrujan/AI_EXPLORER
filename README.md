__Instructions to run__
1. Download the project.
2. Install the supporting packages as far as your operating system is concerned.
    - For instance, I have installed these packages using `pip3` command as shown below.
    - `pip3 install PySide6` ..
3. Navigate to the project folder of AI_EXPLORER.
4. Launch the application from your terminal with the command below.
    - `python3 main.py`

_Supporting packages(so far):_
__PySide6__ - provides access to the complete Qt 6.0+ framework
__matplotlib__ - for creating static, animated, and interactive visualizations
__scikit-learn__ - for machine learning demos
__torch__ - for deep neural networks
__nltk__ - Natural Language Toolkit: for natural language processing
